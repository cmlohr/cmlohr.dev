    What is the BOT_LAB?

    The BOT_LAB is a Discord server community dedicated to building bots of all sorts.  We are a community of developers that build scraper bots, spider bots, chatbots, game bots, trading bots, social media bots, and any other type of bot you can think of.

    Why did you build the BOT_LAB?

    The aim of this community is to teach and to learn about bots.  We believe that bots provide a hands-on, interactive coding experience that is one of the fastest and most engaging ways to pick up new coding skills.  In addition to the fun of creating bots the fact is they are everywhere, even if you don't realize it. Bots today are being used across industires from algorithmic stock trading, the gaming industry, your virtual assistants and many more.

    Testy & Tutor Bot

    Currently, the bot we actively develop the most is Testy.  Testy is a bot used to troubleshoot new features before shuffling the code off to brand new bots.  Testy is already fully configured; this allows us to get straight to the code without creating a new bot every time we get a new idea.

    The crowning jewel of this project is the ongoing development of a coding tutor bot.  You will be able to ask it questions about coding and it will point you in the right direction.  The goal here is to provide real interactive engagement with the users.  Users will get instant answers to their questions, test themselves in quizzes and enjoy project recommendations.

    Let's build together!

    The community is using Replit, Discord and GitHub to develop these bots in public and posting regular updates to GitHub.  We strongly believe in building under the public eye because not only does that make us better engineers but we also might have the opportunity to inspire others to begin coding.
